Рецензія String.h
Автор рецензії - Гордін Михайло
Автор String.h - Накитняк Вадим

Клас String повинен обробляти рядки символів типу char.

Для створення та видалення екземплярів класу Вадим використав Конструктори та Деструктор

Конструктори:

String();
String(const char);
String(const char*);
String(const string&);
String(const String& s, const unsigned multiplayer = 1);

Деструктор:

~String();

Ці конструктори String вміють обробляти різні типи аргументів.

Файл явно поділений на приватну та публічну частину, що може бути плюсом.

Вадим додав методи length(), empty(), clear() для обробки довжини рядка.

size_t length() const;
bool empty() const;
void clear();

Слід зазначити, що Вадим забув додати оператор += для String типу const char, хоча створив для цього типу конутруктор.

String& operator+=(const char*);
String& operator+=(const String&);
String& operator+=(const string&);

Бульові оператори ==,!=,<,<=,>,>= слід було винести в утиліти.

bool operator == (const String&) const;
bool operator != (const String&) const;
bool operator < (const String&) const;
bool operator <= (const String&) const;
bool operator > (const String&) const;
bool operator >= (const String&) const;

Вадим додав утиліти ostream&, для виведення.

ostream operator<<(ostream, const String&);
ostream operator<<(ostream, const String::BadString&);
ostream operator<<(ostream, const String::BadIndex&);

Плюсом є наявність обробки можливих помилок.

Файл String.h у реалізації Вадима мені сподобався, є декілька помилок, але вони некритичні і роботі програми не будуть заважати.
